# Battleship

A browser-based Battleship game where players strategically place ships and battle against a computer opponent or fellow human component.

Built as part of the [The Odin Project](https://www.theodinproject.com/lessons/node-path-javascript-battleship) curriculum to practice test-driven development, game logic architecture, DOM manipulation, and modular JavaScript design.

---

## Features

* Interactive Battleship gameplay
* Randomized computer attacks
* Ship placement validation
* Turn-based attack system
* Hit and miss tracking
* Win and lose game states
* Dynamic game board rendering
* Fully tested game logic with Jest
* Factory functions and modular architecture

---

## Tech Stack

* JavaScript
* HTML
* CSS
* Webpack
* Jest

---

## Game Mechanics

### Player

* Place ships on the board by selecting ship and clicking on board
* Attack enemy coordinates
* Track hits and misses

### Computer AI

* Randomly attacks player board
* Automatically tracks previous moves

### Game Rules

* Ships cannot overlap
* Ships must stay within board boundaries
* First player to sink all enemy ships wins
